1，C语言用头文件可以包含整个文件里的所有内容，包括类，结构体，函数，等等。
	Java里只能导入某个类，或者包里面的一些类。

2，java函数没有默认参数，形如public void fun(int a = 0 )


3，Java里的对象相当于C语言的指针类型，C语言里面多了一种实型。如MyClass class1;C语言能直接实例化
	在Java里只是定义了一个空指针。

4，Java里面的类的成员变量可以直接初始化，但C语言不能，C语言为抽象数据类型。不能直接初始化

5，java没有析构函数，c语言里的析构函数有释放对象内存的操作

6，Java里面的protected关键字很特殊，同包与不同包情况还不一样。

7，Java里有内部类，无多继承，接口能多继承，C语言能多继承。Java继承无二义性，C语言有。

8，C语言里""就是一个字符串常量是一个地址值，而java里的""可以表示一个对象，他的引用也是一个地址

9，C语言里面的空指针，需要人为的赋值，如Tree* tree;tree并不为空,他的内容是0xcccccccc，地址位于栈区的
某个地址，而java里，Tree tree,tree是为空的
