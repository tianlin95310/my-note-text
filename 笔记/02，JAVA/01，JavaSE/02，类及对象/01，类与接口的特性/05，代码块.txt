1，普通代码块

普通方法中的代码块

2，构造块
直接写在类中的代码块，实例化对象的时候才会被调用，并且每次都要调用
构造块先于构造函数执行。

3，静态代码块
在构造块前加上static，静态代码块有静态函数，成员的部分特性
它在类加载的时候执行，且只会执行一次，先于构造块和构造函数执行。
主要用于static类型的初始化
	static
	{
		b = 10;
	}
	static private int a = 10;
	static private int b;

4，同步代码块，主要用于多线程。


5，若有继承情况
子类对象实例化的过程如下

    1，主类的静态块优先级最高
    2，main函数的优先级大于其他类的静态块
    3，父类的静态块大于子类静态块
    4，子类的静态块大于父类的构造块以及构造函数
    5，父类的构造块，构造函数大于子类的构造块，构造函数。
